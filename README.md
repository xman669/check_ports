
 Primero debes instalar esta dependencia 
pip3 install psutil
	1.	Escanea las conexiones activas, mostrando el puerto local, la IP remota, el estado y el proceso asociado.
	2.	Permite cerrar procesos sospechosos ingresando su PID.
	3.	Utiliza psutil.net_connections() para analizar las conexiones en tiempo real.
